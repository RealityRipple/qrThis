# qrThis
Convert any selected text into a QR code.

#### Supports
 * Pale Moon [29.0a1pre - 29.*]

## Building
Simply download the contents of the repository and pack the contents (sans git data) into a .zip file. Then, rename the file to .xpi and drag into the browser.

## Download
You can grab the latest release from the [Official Web Site](//realityripple.com/Software/Mozilla-Extensions/qrThis/).
